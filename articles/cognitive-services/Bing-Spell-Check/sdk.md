# <a name="bing-spell-check-sdk-samples"></a>Bing Spell Check SDK サンプル
Bing Web Search API サンプルには、次のシナリオが含まれています。
1. スペルが間違っているクエリを検索し、応答を解析する。
2. API からエラー応答をトリガーする。

Bing Spell Check SDK は、文脈に応じた文法とスペル チェックを次のプログラミング言語で簡単にアクセスできるようにします。
* [.NET サンプル](https://github.com/Azure-Samples/cognitive-services-dotnet-sdk-samples/tree/master/BingSearchv7)を使ってみる 
    * [NuGet パッケージ](https://www.nuget.org/packages/Microsoft.Azure.CognitiveServices.SpellCheck/1.2.0)
    * 定義と依存関係については、[.NET ライブラリ](https://github.com/Azure/azure-sdk-for-net/tree/psSdkJson6/src/SDKs/CognitiveServices/dataPlane/Language/SpellCheck)も参照してください。
* [Node.js サンプル](https://github.com/Azure-Samples/cognitive-services-node-sdk-samples)を使ってみる 
    * 定義と依存関係については、[Node.js ライブラリ](https://github.com/Azure/azure-sdk-for-node/tree/master/lib/services/spellCheck)も参照してください。
* [Java サンプル](https://github.com/Azure-Samples/cognitive-services-java-sdk-samples)を使ってみる 
    * 定義と依存関係については、[Java ライブラリ](https://github.com/Azure/azure-sdk-for-java/tree/master/cognitiveservices/azure-spellcheck)も参照してください。
* [Python サンプル](https://github.com/Azure-Samples/cognitive-services-python-sdk-samples)を使ってみる 
    * 定義と依存関係については、[Python ライブラリ](https://github.com/Azure/azure-sdk-for-python/tree/master/azure-cognitiveservices-language-spellcheck)も参照してください。

各言語の SDK サンプルには、前提条件と、サンプルのインストール/実行の詳細が記載された ReadMe ファイルが含まれています。
