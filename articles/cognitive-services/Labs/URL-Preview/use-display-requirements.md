# <a name="project-url-preview-use-and-display-requirements"></a>Project URL Preview の使用と表示に関する要件

ソーシャル メディアやチャット ボットなどのサービスでエンド ユーザーが URL を共有した場合に、出典サイトへのハイパーリンクを設定したプレビュー スニペットとサムネイル画像を表示するときは、Project URL Preview から取得したデータのみを使用してください。 Project URL Preview から取得したデータはコピー、保存、キャッシュしないでください。 Web サイトやコンテンツの所有者からプレビューを無効にするよう要請があった場合は、その意向を尊重してください。
