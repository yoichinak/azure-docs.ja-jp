# <a name="azure-maps-traffic-coverage"></a>Azure Maps の交通情報の対象範囲

Azure Maps は、交通の**流れ**と**事故**の形式で豊富な交通情報を提供します。 このデータは、マップ上で視覚化することができます。また、このデータを使用し、実際の運転条件を考慮して、より優れたルートが生成されます。
